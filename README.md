# Git Ninja 🚀

Git Ninja es una herramienta de automatización para optimizar tu flujo de trabajo con Git. Con Git Ninja, podrás gestionar tus commits, ramas y más de forma eficiente, divertida y fácil. ¡Transforma tu forma de trabajar con Git y hazlo como un verdadero ninja! 🥷

## Características principales 🌟

- **Automatización completa** del flujo de trabajo en Git.
- **Revisión rápida de cambios**: añade archivos sin preocuparte por los no rastreados, realiza commit, tags y push en un solo comando.
- **Manejo de ramas**: Asegúrate de estar trabajando en la rama correcta antes de hacer cambios.
- **Interfaz visual divertida** con animaciones y colores para hacerlo más atractivo y fácil de usar.
- **Globalmente accesible**: Se instala globalmente, lo que significa que puedes usarlo en cualquier lugar de tu terminal.

## Instalación 🔧

Para instalar Git Ninja en tu máquina, solo tienes que ejecutar un solo comando con `curl`:

```bash
curl -fsSL https://raw.githubusercontent.com/StivenUrresti/git-ninja/master/install.sh | bash
```

Este comando descargará e instalará Git Ninja globalmente en tu sistema.

> **Nota:** Si no tienes permisos de administrador, ejecuta el script con `sudo`:

```bash
sudo curl -fsSL https://raw.githubusercontent.com/StivenUrresti/git-ninja/master/install.sh | bash
```

## Uso 🥷

Una vez instalado, puedes usar Git Ninja directamente desde tu terminal.

### Ejemplos de comandos:

- **Iniciar el flujo de trabajo de Git Ninja:**
    ```bash
    git-ninja
    ```

  Esto iniciará el flujo de trabajo, revisando tu repositorio, añadiendo archivos no rastreados, realizando commit y push de cambios.

- **Obtener ayuda:**
    ```bash
    git-ninja --help
    ```

  Te mostrará todas las opciones disponibles para personalizar tu flujo de trabajo.
